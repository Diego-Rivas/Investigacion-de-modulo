¿Qué es el operador MOD?

El MOD, es como una herramienta matemática que permite obtener el residuo de una división entre dos números. En otras palabras, nos dice cuánto "sobra" después de realizar la división.

¿Para qué sirve el operador MOD?

  • Determinar si un número es par o impar.
  • Devuelve el residuo de una división entre dos números.
  • Para probar si dos números pueden dividirse de manera exacta.
